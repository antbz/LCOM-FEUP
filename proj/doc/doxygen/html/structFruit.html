Detailed Description

Sprite-like struct used for displaying fruits and other elements.

Contains the images, positional and contextual information useful for drawing objects on the screen. Used for buttons, fruits and the mouse pointer. Contains two images to allow for easy switching between pressed down/released mouse/button states and whole/cut fruit states. The boolean members are used to decide which image to display and whether the object's position should be updated or not.
